## 게임 플레이 방식
* 각 지역들을 다니면서 지역마다 있는 퀘스트를 클리어

  * 퀘스트가 없는 지역은 다른 상호작용이 있음
* 먼저 하고싶은 퀘스트를 골라 할 수 있다.

  * 무슨 퀘스트를 먼저 하는지에 따라 다른 엔딩
## 팀원 및 역할
* **유종원(팀장)**
  * 코드 작업
  * 의견 조율
  * 발표 자료 제작 밑 검토
* **김재현**
  * UI작업
  * 전체적인 그래픽
* **김시언**
  * 팀원들에게 "채보를 재미있게 짜는법" 공유
  * 리듬게임 채보 제작
* **김창율**
  * 리듬게임 채보 제작
  * 코드 작업 보조
* **이호진**
